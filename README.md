# -Institution-Performance-Students-Analytics-Dashboard
Interactive Power BI dashboard analyzing institution performance, student enrollment trends, course-wise revenue, and age-group demographics. Features KPI cards, slicers, bar charts, and donut charts to provide actionable insights using Power BI, Excel, and DAX.
Institution Performance & Student Analytics Dashboard

Key Features

- KPI Cards for Total Fees, Total Students, and Average Students Per Year
- Monthly Enrollment Analysis
- Course-wise Revenue Analysis
- Age Group Distribution
- Interactive Filters (Year, City, Course Name)

Tools Used

- Power BI
- Excel
- DAX

Dashboard Preview (Dashboard_image.png)

Key Insights

- Davinci Resolve generated the highest revenue
- 21–29 age group represents the largest student segment
- November showed the highest enrollment
